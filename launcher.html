✨ Νέα Χαρακτηριστικά

  • Modern iPhone-style UI με animations
  • Πλήρως responsive για όλες τις συσκευές
  • PWA support - Εγκατάσταση σαν native app
  • Offline mode - Λειτουργεί χωρίς internet
  • Swipe gestures για mobile navigation
  • Persistent settings με localStorage

🎯 Γρήγορη Έναρξη

  • Ανοίξτε με web server (VS Code Live Server)
  • Δοκιμάστε το test.html για έλεγχο
  • Δοκιμάστε το test-sunrise.html για ανατολή/δύση
  • Προσαρμόστε τις ρυθμίσεις (⚙️)
  • Εγκαταστήστε στο mobile (Add to Home Screen)
